# vscode-reminder

> 一款定时提醒您休息和饮水的插件

### 自定义配置

以mac平台为例，键入 `Cmd + ,` 打开配置页可以自定义提醒间隔时间和提醒大图

![](img/setting.png)

### 使用&预览

默认每间隔1小时弹出提醒，或者手动键入 `Cmd + shift + p` 输入reminder指令可弹出提醒

![](img/review.png)
